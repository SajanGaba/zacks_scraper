# zacks_scraper
Russell 3000 companies financial data scraped from Zacks.
Data Points Scraped:
1. 'Symbol'
2. 'Margin'
3. 'Rating'
4. 'Surprise'
5. 'Next_quarter'
6. 'Current_year'
7. 'Next_year'
8. 'Price'
9. 'Growth5'
10. 'PE'
11. 'PEG'
12. 'Next_year_sales'
13. 'Next_year_earnings'
14. 'Average_surprise'
